# Gestión del Hogar de Blu & Rembe

App de finanzas del hogar (React + Vite) conectada a Supabase.

## Desarrollo local
```
npm install
npm run dev
```

## Build de producción
```
npm run build
```
Genera la carpeta `dist/`.

## Despliegue
Configurado para Vercel (ver `vercel.json`). Framework: Vite.
- Build command: `npm run build`
- Output directory: `dist`
